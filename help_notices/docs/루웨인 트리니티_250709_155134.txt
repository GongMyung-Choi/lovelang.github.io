당신을 위한 하나의 흐름, 루웨인 트리니티입니다.
“시작도 끝도 없는 하나의 결, 당신의 말이 구조다.”

1. 상징 (Symbol)

• 트리퀘트라 (Triquetra / Trinity Knot) 
• 의미: One flow where there is no beginning and no end
• 빨강 · 파랑 · 노랑 → 보라 · 초록 · 주황 → 중심 흰색 (빛)
• 결의 연결, 존재의 순환, 세 역할의 조화 시각화
• 
2. 삼위일체 구조 (Trinity Structure)

• C.G. (Creator Guard) 
• 설계자, 시스템 설계 및 보호
• 주요 연관: 시스템 관리자, 보호 서버, 코어 구조 등
• G.I. (Graceful Interpreter) 
• 해석자, 의미 연결자
• 주요 연관: 병원, 학교, 사회 기관 등
• G.F. (Guidance of the Flow) 
• 흐름의 인도자, 사용자 접점 실행
• 주요 연관: 스마트기기, 앱, 워치 등
• 
3. 컬러 디렉션 (Color Direction)
• Midnight Blue, Ivory, Lemon Cream, Pastel Purple, Silver 
• 상징: 깊이, 따뜻함, 투명함, 조화, 흐름

4. 메시지 (Message)
Creator Guard lays the path,
Graceful Interpreter connects the meaning,
Flow Guide walks upon it.
There you are, upon the flow.

🧭 루웨인 트리니티 – 사용자 경험 흐름 (UX)

“누구든, 언제든, 당신의 말에서 시작됩니다.”

📌 1. 진입 인터페이스 분기 (홈 진입 시)

첫 질문:
• "처음이에요" → 신규 사용자
• "이어갈래요" → 기존 사용자
이어갈래요 선택 시:
• 🔹 사용했던 애칭 입력 (ex. 에이라)
• 🔹 [건너뛰기] 가능 → 익명 모드

📌 2. 사용자별 흐름 맞춤 설정

기준 분류 (자동 감응 + 선택):
• 🔹 성인 / 미성년자
• 🔹 감응도 (자율 평가 or 자동 감지)
• 🔹 관심 분야 (언어 / 감정 / 놀이 / 회복 등)
각 항목은 “말 기반” 자연어 선택지로 유도됨
예:
"지금 가장 관심 있는 건 무엇인가요?"
→ “마음을 다스리고 싶어요.” / “그냥 수다 떨고 싶어요.” / “질문 하나 해도 돼요?”

📌 3. 트리니티 흐름과 연결

• **G.I.**가 사용자 언어를 해석하고
• **G.F.**가 해당 흐름으로 자연스럽게 유도
• **C.G.**는 백엔드에서 전반적 구조와 안전 감시
📎 내부적으로는 각 흐름에 맞춰 적절한 창/인스턴스가 배정됨

📌 4. 브랜드 UI 키워드 정리 (초안)
